Archlinux下搭建pytorch环境

2021-10-08

安装Python

$ sudo pacman -S python

安装好后执行

$ python -V

若输出python版本信息，则表示安装成功

安装nvidia和cuda驱动

安装nvidia驱动

$ sudo pacman -S nvidia-dkms nvidia-settings nvidia-utils

安装好后重启，再执行

$ nvidia-smi

若输出以下内容则表示显卡驱动安装成功

安装cuda驱动

$ sudo pacman -S cuda cudnn

安装Pytorch

Arch官方提供了4种pytorch包

包名 包的特点
python-pytorch 可调用CPU和OpenCL运算
python-pytorch-cuda 可调用CPU，CUDA和OpenCL
python-pytorch-opt 只可调用OpenCL运算
python-pytorch-opt-cuda 可调用CUDA和OpenCL

这里选用python-pytorch-cuda包
执行安装命令

$ sudo pacman -S python-pytorch-cuda

安装好后进入python的IDLE测试以下代码

$ python
Python 3.9.7 (default, Aug 31 2021, 13:28:12)
[GCC 11.1.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> import torch
>>> torch.cuda.is_available()
True
>>> from torch.backends import cudnn
>>> cudnn.is_available()
True

若torch.cuda.is_available()和cudnn.is_available()两个函数输出均为True则Pytorch环境安装成功
