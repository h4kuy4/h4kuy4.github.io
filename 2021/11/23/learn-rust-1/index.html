2021-11-23

安装开发环境

安装rustup和cargo。

# pacman -S rustup

cargo将会作为rustup的依赖被安装。
rustup是rust官方提供的工具链管理工具。
cargo是rust的项目构建工具和包管理器。

安装toolchain(linux环境)

$ rustup install stable-x86_64-unknown-linux-gnu

安装测试

$ rustc -V
rustc 1.56.1 (59eed8a2a 2021-11-01)
$ cargo -V
cargo 1.56.0 (4ed5d137b 2021-10-04)

若正常输出版本信息则安装成功。

Hello world

main.rs

fn main() {
    println!("Hello world!")
}

编译

$ rustc main.rs
$ main
Hello world!
