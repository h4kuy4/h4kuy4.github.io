Archlinux下搭建pytorch环境

2021-10-08

安装Python $ sudo pacman -S python 安装好后执行 $ python -V 若输出python版本信息，则表示安装成功 安装nvidia和cuda驱动 安装nvidia驱动 $ sudo pacman -S nvidia-dkms nvidia-settings nvidia-utils 安装好后重启，再执行 $ nvidia-smi 若输出以下内容则表示显卡驱动安装成功 +-----------------------------------------------------------------------------+ | NVIDIA-SMI 470.74 Driver Version: 470.74 CUDA Version: 11.4 ..
